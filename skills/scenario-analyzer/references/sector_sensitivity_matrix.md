# Sector Sensitivity Matrix

This reference organises sector sensitivity to various event types in matrix form. Consult it during
scenario analysis to quickly identify which sectors are likely to be affected.

## Legend

**Impact:**
- `++` : strong positive
- `+` : positive
- `0` : neutral / minor
- `-` : negative
- `--` : strong negative

**Confidence:**
- `H` : High (past patterns consistent)
- `M` : Medium (context-dependent)
- `L` : Low (high uncertainty)

## How to Use

1. **Identify the event type** from the headline.
2. **Look up the matching matrix** above.
3. **Read the impact and confidence levels** by sector.
4. **Use example tickers as starting points** for deeper analysis.
5. **Combine multiple matrices** for compound scenarios.

## Caveats

- **Individual stocks may diverge from their sector** view.
- **Distinguish immediate vs delayed effects** when applying the matrix.
- **Magnitude matters** — impacts scale with event size.
- **What's priced in** — reactions are limited if the market is already discounting the event.
